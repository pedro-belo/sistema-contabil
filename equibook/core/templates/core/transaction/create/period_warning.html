Configuração do Período Contábil

Para cadastrar uma transação de débito ou crédito, é necessário configurar o período contábil correspondente.

O período contábil determina o intervalo de tempo para o qual as transações financeiras serão registradas. Certifique-se de que o período contábil esteja corretamente configurado antes de prosseguir com o cadastro das transações.
